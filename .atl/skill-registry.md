# Skill Registry — UK Dropshipping Hub

This registry defines trigger-matched agent skills and conventions for this codebase.

## Project Standards

### TypeScript & NestJS Styling
- Use async/await explicitly. Always declare return types for controllers and services.
- Follow the controller-service-repository pattern using NestJS Dependency Injection.
- Group REST resources logically in modules.

### React Router v7 & Styling
- Maintain the unified brand dark (`#1a1a1c`) and brand accent (`#8b5cf6`) color palette.
- Keep components responsive and clean using TailwindCSS v4.
- Use explicit TypeScript interfaces for loader and action payloads.
